No Wall Repair 
====================

Disable repair for walls. This will prevent bots from sitting on walls 'repairing' them and taking damage / dying needlessly.

Updated for Factorio 2.1.

Original mod by Pawz777:
https://github.com/Pawz777/no-wall-repair

This reassembled version updates the mod metadata and runtime script for Factorio 2.x compatibility while keeping the original behavior.
